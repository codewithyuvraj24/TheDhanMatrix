# Contributing to Dhanmatrixcapital

Thank you for your interest in contributing to Dhanmatrixcapital! We are building a high-performance wealth management platform and welcome your contributions.

## Code of Conduct

This project and everyone participating in it is governed by the [Dhanmatrixcapital Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.

## How to Contribute

### Reporting Bugs

Before creating bug reports, please check existing issues to ensure the problem hasn't already been reported. When creating a bug report, please include as many details as possible:

* Use a clear and descriptive title.
* Describe the exact steps which reproduce the problem.
* Provide specific examples to demonstrate the steps.
* Describe the behavior you observed after following the steps.
* Explain which behavior you expected to see instead and why.

### Suggesting Enhancements

Enhancement suggestions are tracked as [GitHub issues](https://github.com/yourusername/TheDhanMatrix/issues). When creating an enhancement suggestion, please include:

* A clear and descriptive title.
* A detailed description of the proposed enhancement.
* Examples of how the enhancement would benefit the project.

### Pull Requests

1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Make sure your code lints.
5. Create the Pull Request.

## Development Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables (see `.env.local.example`)
4. Run the development server: `npm run dev`

## Style Guide

* We use TypeScript for all new code.
* We follow functional programming patterns where possible.
* We use Tailwind CSS for styling.
* Commit messages should be clear and descriptive.

Thank you for contributing!
